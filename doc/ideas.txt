Генерируется ResourceLocator
    Генерируется конструктор, где создаются все провайдеры всех ресурсов.
        dependencies провайдеров передаются через конструктор этих провайдеров
        провайдер всегда существует в единственном экземпляре
        провайдер может предоставлять различные ресурсы
            - обычные ресурсы - классы, написанные пользователем
            - callers - специальные автоматически сгенерированные классы для вызова конкретного метода конкретного ресурса
        код провайдера
            - может уже существовать (предоставлен пользователем),
            - может быть автоматически сгенерирован
            - может быть сгенерирован генератором провайдера (который предоставлен пользователем)
                    в этом случае генератор провайдера получает информацию о параметре, для которого провайдер будет создавать ресурс
    В локаторе генерируется геттеры для каждого ресурса
        геттер берет нужный провайдер и вызывает метод для получения нужного ресурса
